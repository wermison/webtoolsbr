Como é feito o cálculo?

Basta somar a frente do terreno e fundos. Ex: Frente = 20,00 Fundos = 32,00 Somatória divide-se por (2) 52,00/2 = 26,00 Após obter esse resultado Soma-se os lados Ex: Lado direito = 41,00 Lado esquerdo = 53,00 Somatória divide-se por (2) = 94,00/2 = 47,00 Após obter esse resultado Multiplicar os dois totais Ex: 26,00 X 47,00 = 1.222,00 M2 ( área do terreno).

O que é um Trapézio Escaleno.

Um terreno com os 4 lados diferentes é uma forma geométrica chamada de trapézio escaleno. O trapézio é classificado como escaleno quando não possui nenhum de seus lados congruentes, ou seja, todos os lados têm medidas diferentes e, consequentemente, todos os ângulos internos têm medidas diferentes.
